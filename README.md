### Installation

```
$ git clone https://github.com/fitztrev/trevdev.git ~/trevdev
$ cd ~/trevdev
$ make install
```

##### Configuring iTerm2

1. Set iTerm2 preferences folder to `~/trevdev/iterm2`
2. Restart iTerm2

##### Generate a new SSH key

```
$ make ssh-key
```

### Updating

```
$ make update
```
